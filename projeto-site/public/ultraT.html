Ultra T - Projeto

O ultra-T é um software de controle de computadores da empresa, com um dashboard intuitivo, planejado para ajudar funcionarios da infraestrutura com funcionalidades como: geramento de logs personalizados, geração de relatorios, chat interno com os funcionarios da empresa, avisos e alertar para evitar erros ou riscos.
